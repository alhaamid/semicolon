PROBLEM STATEMENT LEADERBOARD

Rabbiya is a great individual but is also always thinking of ways to annoy her friends. So she invites n of her friends (numbered 1, 2, ... n) with Obsessive-Compulsive Disorder to her home to play LUDO. Each friend should belong to exactly one group, and each group must have at least one friend in it.

Each friend i is willing to participate only if the group the friend belongs to has exactly T[i] friends in it. Rabbiya wants to make sure that everyone participates, but she is unsure if that is possible. Help Rabbiya in her evil plan and figure out if there is a way to assign friends to groups such that everyone is happy with the size of their group. If it is possible, display the number of groups needed.If it is impossible, display -1.

Input: You will read only one file named: "q1.in". You should assume that this file is in the same directory as your code for this question. The format for the file is as follows:

  • The first line will contain one number n which is the number of test cases
  • n lines follow
  • Each of the next n lines will contain a unique T. Thus, each of the next n lines will contain an array of numbers which will be comma (,) separated

    • Output: For each of the n T's, you need to display one and only one number. Each of these numbers should be on a new line. So in short, your output should contain n numbers with each number number separated by a newline(\n) character.

    Constraints:

  • T will contain between 1 and 500 elements, inclusive
  • Each element in T will be between 1 and 500, inclusive

    • Sample Input:
    4
    1,2
    2,2
    3,3,2,2,2
    2,3,2,3,2,3,2,1,1,2,2

    Sample Output:
    -1
    1
    -1
    6
